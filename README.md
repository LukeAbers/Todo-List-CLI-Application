# Todo List CLI Application

A simple command-line interface (CLI) tool for managing a to-do list. Easily add, remove, and view tasks.

## Features

- **List tasks**: Display all the tasks with their respective indexes.
- **Add tasks**: Append tasks to the end of the list.
- **Remove tasks**: Delete a task based on its index.
- **Persistent Storage**: Tasks are saved to a file, so they persist between program runs.

## Installation and Usage

1. **Clone the repository**

2. **Run the application**:
   ```bash
   python todo.py
   ```
3. **Follow the on-screen prompts** to manage your tasks.
